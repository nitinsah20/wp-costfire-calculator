# WP Costfire Calculator

A WordPress plugin to calculate Cost of Fire with a simple and user-friendly interface.

## Features
- Easy shortcode integration
- Clean UI
- Accurate calculation logic
- Lightweight and fast

## Installation
1. Upload plugin folder to /wp-content/plugins/
2. Activate plugin from WordPress admin
3. Use shortcode to display calculator : [coastfire_calculator]
